# maya-scripts

A collection of scripts for [Autodesk Maya](https://www.autodesk.com/products/maya/) — Python and MEL tools for automating modeling, rigging, animation, and pipeline tasks.

## Usage

Copy the scripts into your Maya scripts directory, or add this repository's path to your `MAYA_SCRIPT_PATH` / `PYTHONPATH`.

Typical script locations:

- **Windows:** `%USERPROFILE%\Documents\maya\<version>\scripts`
- **macOS:** `~/Library/Preferences/Autodesk/maya/<version>/scripts`
- **Linux:** `~/maya/<version>/scripts`
